## 確認方法

### ブラウザの開発者ツールで確認

1. **Elementsタブ**: HTMLの構造を確認
2. **Stylesタブ**: 適用されているTailwindクラスを確認
   - 例: `.bg-primary-500`, `.text-text-primary`, `.min-h-screen`
3. **Networkタブ**: `tailwind.css`が正しく読み込まれているか確認

### コンソールで確認

```javascript
// Tailwind CSSが読み込まれているか確認
getComputedStyle(document.querySelector('.bg-primary-500')).backgroundColor
// 期待値: "rgb(148, 128, 237)" または類似の値
```

## トラブルシューティング

### スタイルが適用されない場合

1. **CSSファイルの確認**
   ```bash
   ls -lh tailwind.css
   # ファイルが存在し、サイズが0でないことを確認
   ```

2. **生成されたCSSの確認**
   ```bash
   grep -c "bg-primary-500" tailwind.css
   # 1以上が返れば正常
   ```

3. **ブラウザのキャッシュクリア**
   - ハードリロード（Cmd+Shift+R / Ctrl+Shift+R）

4. **開発サーバーの再起動**
   ```bash
   npm run dev
   ```
